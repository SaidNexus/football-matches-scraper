# 🏆 أداة لاستخراج بيانات المباريات

يستخرج هذا المشروع **أسماء البطولات، أسماء الفرق، النتائج، وتواريخ المباريات** من موقع ويب ويحفظها في **ملف CSV**.

## 📌 المتطلبات  
قبل تشغيل السكريبت، قم بتثبيت المكتبات المطلوبة عبر الأمر التالي:  
```bash
pip install -r requirements.txt
```

## 🚀 كيفية التشغيل  
1️⃣ قم بتشغيل السكريبت باستخدام:  
```bash
python yallakora.py
```
2️⃣ أدخل **التاريخ المطلوب** بالصيغة **MM/DD/YYYY** عند الطلب.  
3️⃣ أدخل اسم الملف، وسيتم حفظه في المجلد (outputs)

## 📊 البيانات المستخرجة  
- 🏆 **اسم البطولة**  
- ⚽ **الفريق الأول**  
- ⚽ **الفريق الثاني**  
- ⏰ **وقت المباراة**  
- 🔢 **النتيجة**  

## 📁 ملف الإخراج  
بعد تشغيل السكريبت، سيتم إنشاء **ملف CSV** يحتوي على جميع المباريات المتاحة.  

## 🔍 مثال للبيانات المستخرجة:  
| البطولة            | الفريق الأول  | النتيجة | الفريق الثاني   | وقت المباراة |
|---------------------|--------------|---------|----------------|--------------|
| الدوري الإنجليزي   | ليفربول       | 2 - 1   | مانشستر سيتي   | 20:00        |
| دوري الأبطال       | ريال مدريد    | 3 - 0   | بايرن ميونخ    | 22:00        |

---

## 🛠️ الأدوات المستخدمة  
- `requests` لجلب محتوى صفحة الويب  
- `BeautifulSoup` لتحليل واستخراج البيانات من HTML  
- `csv` لحفظ البيانات في **ملف CSV**  
- `os`

## 📝 ملاحظات  
- تأكد من أن **الموقع المستهدف يسمح باستخراج البيانات (web scraping)** عن طريق التحقق من سياسة **robots.txt** الخاصة به.  
- يمكن تعديل السكريبت ليعمل مع مواقع أخرى أو لاستخراج بيانات إضافية مثل **إحصائيات المباراة، تشكيل الفرق، تفاصيل الأهداف، وغيرها**.
